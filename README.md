## 📌 Project Overview

This project demonstrates a complete DevOps pipeline for deploying a Machine Learning model using Flask and CI/CD practices.

## ⚙️ Features

* Machine Learning model using scikit-learn
* REST API built with Flask
* Deployment using ngrok
* Version control using GitHub
* CI/CD pipeline using GitHub Actions

## 🔗 API Endpoint

POST /predict

## 📥 Input Example

[5.1, 3.5, 1.4, 0.2]

## 📤 Output Example

{"prediction": 0}

## 🛠️ Technologies Used

* Python
* Flask
* GitHub Actions
* Google Colab

## 📊 Results

The model successfully predicts outputs via API and demonstrates automated deployment workflow.

## 🎯 Conclusion

This project shows how DevOps practices can streamline AI model deployment efficiently.
